# 정렬 알고리즘 비교

## 정렬 알고리즘 비교 표

|    Sort   	|  Best 	|  Worst  	| Stable 	|  Memory  	|
|:---------:	|:-----:	|:-------:	|:------:	|:--------:	|
| 버블 정렬 	|   N   	|   N^2   	|  TRUE  	|     1    	|
| 선택 정렬 	|  N^2  	|   N^2   	|  FALSE 	|     1    	|
| 삽입 정렬 	|   N   	|   N^2   	|  TRUE  	|     1    	|
|  셸 정렬  	| NlogN 	| Nlog^2N 	|  FALSE 	|     1    	|
| 합병 정렬 	| NlogN 	|  NlogN  	|  TRUE  	|     N    	|
|  퀵 정렬  	| NlogN 	|   N^2   	|  FALSE 	| logN ~ N 	|
